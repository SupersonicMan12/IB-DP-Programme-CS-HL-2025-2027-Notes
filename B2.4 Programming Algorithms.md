## Bubble Sort and Selection Sort (3)

Bubble Sort:
$n^2$ swaps to sort
```java
int[] numbers = {7,3,2,9,1};
for (int i = 0; i < numbers.size(); i++){
	for (int j = 0; j < number.size()-1; j++){
		if (numbers[j] > numbers[j+1]){
			int temp = numbers[j];
			numbers[j] = numbers[j+1];
			numbers[j+1] = temp;
		}
	}
}
```

Selection Sort:
Building the final list from left to right
```java
int[] numbers = {7,3,2,9,1};
for (int i = 0; i < numbers.size()-1; i++){
	int idx = 0;
	for (int j = i+1; j < numbers.size()-1; j++){
		if (numbers[j] < numbers[idx]) idx = j;
	}
	int temp = numbers[i];
	numbers[i] = numbers[idx];
	numbers[idx] = temp;
}
```

## Recursive Algorithms (5)
Recursive Algorithms = a problem-solving method where a function calls itself, either directly or indirectly, to solve smaller instances of the same problem until it reaches a base case that stops the recursion.
IB tests simple, **non-branching recursive algorithms** in programming only

Examples:
- Fibonacci Numbers: $F_n = F_{n-1} + F_{n-1}$
- Tower of Hanoi

Practice Writing:
- Sum of $1+2+\cdots+n$
- Sum of $1^k+2^k+\cdots+n^k$
