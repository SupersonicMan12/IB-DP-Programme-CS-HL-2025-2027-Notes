### A2.1.1 Describe the purpose and characteristics of networks. 
- Networks: local area network (LAN), wide area network (WAN), personal area network (PAN), virtual private network (VPN)

Network = a way of connecting computers so they can:
- share software, hardware, and communicate
- computers or peripherals = nodes

**LAN** Local Area Network = connects nodes over a single building or site
- Electric cable / short-range radio signals
- Wifi

**WLAN** Wireless Local Area Network = connects nodes over a single building or site, without wire
- Modern Wifi = IEEE 802.11 standards

**VLAN** Virtual Wireless Local Area Network = network of computers that behave as if they are connected to the same topology, but are actually physically located in different segments of a LAN.
- Maintained by software
- Flexible

**WAN** - Wide Area Network operate over much larger distances than LAN's
- Long range communication systems, e.g. the Internet
- Provided by common carriers
- Switched: Temporary connection, e.g. public telephone
- Dedicated: Permanent connection, e.g. leased line

**PAN** - Personal Area Network = computer network used for data transmission among devices such as computers, telephones and personal digital assistants.
- Literally on a personal level

**VPN** - Virtual Private Network = extends a private network across a public network, such as the Internet.
- establishing a virtual point-to-point connection through the use of dedicated connections, virtual tunneling protocols, or traffic encryption.
- **Authentication** allows VPN clients and servers to correctly establish the identity of people on the network. 
- **Encryption** allows potentially sensitive data to be hidden from the general public.

### A2.1.2 Describe the purpose, benefits and limitations of modern digital infrastructures. 
- Modern digital infrastructure: the internet, cloud computing, distributed systems, edge computing, mobile networks 
- Examples where specific networks are used may include the worldwide web (WWW), cryptocurrency blockchains, smart traffic lights, a school. 

**Internet**:
- The internet is a global network of interconnected devices (routers, servers, computers) which exchange data using standardized protocols. 
- The worldwide web (WWW) uses the internet as infrastructure to manage a global network of hypertext transfer protocol (HTTP)-based resources.

**Cloud Computing:**
- Network of **remote** servers accessible via the internet and used to store, manage and process data, including offering software.

**Distributed System**:
- Network of densely connected independent computers that appear to the users of the system as a single coherent system.
- Efficient and reliable

**Edge Computing**:
- Computation on the peripheral or the source of the information before uploading to servers
- Reduces long-distance physical data transfer

**Mobile Networks**:
- Allows for the connectivity of mobile devices
- Mostly cellular


### A2.1.3 Describe the function of network devices.
- Gateways, hardware firewalls, modems, network interface cards, routers, switches, wireless access points 
- How devices map to the layers of the TCP/IP model 

**Network Interface Card** 
- Hardware component that allows connection to the network
**Hub**
- Repeater, amplifying signals to networked devices, effectively broadcast
**Switch**
- Similar to a hub but with ability to filter out irrelevant traffic, while doing 1-to-1 correspondence
**Gateway**
- Gate / Connection between two networks
- Translates network protocol
**Router**
- Translate and direct traffic between networks
**Modem**
- Modulator-demodulator
- Converts between computer's digital data and analog sound
**Hardware Firewall**
- Denies incoming network traffic based on predetermined set of security rules
**Wireless Access Point**
- Extenders for wireless signals
**Bridge**: not important

**Network Models**:
- Client-server model: client requests network service, server provides it (central command)
	- Benefits: centralized control, scalability, efficiency, security
	- Drawbacks: single point of failure, risk; cost at the center
	- Email, online banking
- Peer-to-peer model
	- Benefits: decentralized risk, cost effective, scalability, direct sharing
	- Coordination (e.g. synchronization), reliability

### A2.1.5 Describe the function of the TCP/IP model. (HL only) 
- Four layers of TCP/IP Model: Application, Transport, Internet, Network Interface 
- The role of each layer and the interaction between these layers to ensure reliable data transmission over a network

The TCP/IP Model is all about protocols!!

**Application Layer**, protocols that applications to exchange data:
- HTTP, SMTP, DNS

**Transport Layer**: communication between hosts:
- TCP (Transmission Control Protocol), reliable, ordered delivery
- UDP (User Datagram Protocol), faster but less reliable

**Internet Layer**:
- Mainly through IP (Internet Protocol)

**Network Interface Layer**:
- Data framing, MAC addressing.
- Physical transmission of data

The four layers interact: 
1. I send email with SMTP
2. TCP carries my email to a server
3. Internet Layer delivers via IP
4. Network Interface does the physical transmission of data, inter-continent...
